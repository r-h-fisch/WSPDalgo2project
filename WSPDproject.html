DEFINITION OF sWSPD

Δu = diameter of node u
Δv = diameter of node v
▢u = set of leaves in node u
▢v = set of leaves in node v
F = created s-WSPD
First call of the function is: WSPD(root, root)

WSPD(u, v)
{
  if ∆u < ∆v
    swap(u, v)
    if ∆u ≤ sd(▢u,▢v)
     F = F ∪{(u,v)}
  else
    for each child w of u 
      WSPD(w, v)
}
